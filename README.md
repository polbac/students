## Requeriments

- node > 10
- yarn

## Install

```
yarn install
```

## Dev

```
yarn run dev
```
## Build

```
yarn run build
yarn run build:dev # for development
```

## Start Application

```
yarn run start
yarn run start:dev # for development
```

## Deploy

```
now
```

and next

```
now --prod
```


## Stack

- nextjs
- react
- formik
- redux & redux-thunk (only for dashboard)
- typeorm
- material-ui
- yup
